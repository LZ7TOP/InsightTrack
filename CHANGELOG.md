# 📜 InsightTrack 版本变更日志 (Changelog)

本项目所有重要变更均记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。

---

## 🚀 [v1.1.0] - 2026-07-22

### ✨ Added (新增)

- **官方网站与仓库配置关联**：项目全局集成官方主页地址 `https://insighttrack.lz7.com.cn` 与 GitHub 开源仓库入口，更新 `manifest.json` 与 `package.json` 的 homepage 规范。
- **单站下钻与跨段比对 Header 大图标重构**：引入 `FaviconImg` 大尺寸网站 Icon（`w-14 h-14 rounded-2xl shadow-sm`）、网站名称标题与蓝框域名 Badge 胶囊，全维度对齐视觉审美。
- **Popup 最常访问媒体列表**：重构插件 Popup「今日最常访问」列表为媒体对象 Flex 排版，最左侧独占独立图标列，右侧第一行展示网站名称与域名及活跃时长，第二行展示全宽比例进度条。

### 🔄 Changed (优化)

- **概览分析图表与布局重构**：
  - 将原单图全量网站使用时长对比柱状图拆分为两个独立的横向条形图（实际活跃时间对比 vs 挂机/后台驻留时间对比），纵轴展示网站标题，柱右展示域名及具体时长。
  - Hover 悬浮卡片图标由 Emoji 升级为工程化内联 SVG 图标。
  - 移除「每日整体时间起伏趋势」与「Top 10 网站占比」图表，重构为「网站使用明细列表」，限制列表最大高度为 360px 支持垂直滚动，吸顶固定 Sticky 表头；实际活跃时间与总驻留时间单元格采用上下结构展示（上方展示时长与占比，下方展示全宽比例进度条）。

## 🚀 [v1.0.0] - 2026-07-22

### ✨ Added (新增)

- **智能双维度时间追踪**：毫秒级精准区分浏览器窗口前台焦点交互（实际活跃时间）与后台标签页挂机总时长（页面驻留时间）。
- **无操作空闲检测开关**：支持自定义开启/关闭无操作检测（超时 30s ~ 10min 自动暂停活跃计时）。
- **全维度数据可视化看板**：
  - **双色嵌套胶囊对比柱状图**：外层柔和底柱表示总驻留时间，内层电光蓝渐变柱表示实际活跃时间。
  - **每日起伏趋势折线图**与 **24 小时活跃时段分布热力柱状图**。
  - **Top 10 域名占比饼图**与**全量网站访问明细排行榜**。
- **单站点深度下钻**：支持查看特定域名的 URL 级页面访问统计及全量切页流水日志溯源。
- **历史跨段比对**：针对选定站点，按选择周期（今日 / 近 7 天 / 近 30 天 / 自定义区间）开展逐日对比。
- **自定义时间区间选择器 (`DateRangePicker`)**：预设快捷按钮 + 双日期 Picker 弹窗，同一天智能识别 `00:00:00 - 23:59:59`。
- **吸顶控制与折叠侧边栏**：全局固定吸顶 Header，侧边栏支持平滑收折为极简图标状态。
- **项目介绍页面与 4 字标准导航**：在侧边栏底部集成「项目介绍」面板。
- **偏好设置与数据管理**：支持自定义黑名单、子域名归并、每日专注目标健康提醒、自动刷新频率及 JSON 数据全量备份导出/恢复。
- **工程化流水线**：配置 ESLint v9、Prettier、Commitlint、Husky pre-commit 钩子、`scripts/package.js` 自动化打包与 GitHub Actions CI/CD 流水线。

### 🔒 Security (安全与隐私)

- **100% 本地隐私存储**：所有切页日志与访问数据完全存储在本地 IndexedDB 数据库中，零服务端依赖，绝不上报云端。
