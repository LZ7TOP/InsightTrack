## ✨ 核心更新亮点

### 1. 概览分析图表与使用明细表重构

- **双横向对比柱状图**：将原单图全量网站使用时长对比柱状图拆分为两个独立的横向条形图（实际活跃时间对比 vs 挂机/后台驻留时间对比），纵轴展示网站标题，柱右展示域名及具体时长。
- **SVG 内联矢量图标**：Hover 悬浮卡片图标全量升级为工程化内联 SVG 图标。
- **网站使用明细列表**：移除「每日整体时间起伏趋势」与「Top 10 网站占比」图表，重构为「网站使用明细列表」，限制列表最大高度为 `360px` 支持垂直滚动，吸顶固定 Sticky 表头；实际活跃时间与总驻留时间单元格采用上下结构展示（上方展示时长与占比，下方展示全宽比例进度条）。

### 2. 深入细节的单站下钻与跨段比对 Header 重构

- **大尺寸 Favicon 矢量 Icon**：引入 `w-14 h-14 rounded-2xl` 大尺寸网站图标，配合圆角与阴影。
- **网站名称与域名 Badge 胶囊**：主标题展示粗体网站名称，旁边搭配蓝色边框与背景的 `domain` 域名标签。
- **全维度 3 大单站核心指标卡片**：展示单站实际活跃时间、单站总驻留时间与单站专注率。

### 3. 扩展 Popup「今日最常访问」媒体对象排版

- **图标最左侧独立列**：把 Favicon 图标单独提取为 Flex 的第一个独立 Element，垂直居中独占最左侧。
- **右侧双层结构**：第一行展示网站名称与域名及活跃时长，第二行展示全宽 rounded 比例进度条。

### 4. 官方主页与工程规范

- 绑定官方主页地址 `https://insighttrack.lz7.com.cn`，同步更新 `manifest.json` 与 `package.json` 中的 `homepage_url` 配置。
