# 🚀 InsightTrack v1.0.0 首个正式版本发布

🎉 **InsightTrack v1.0.0** 正式发布！InsightTrack 是一款由 **LZ7工作室** 开发的现代 Chrome / Chromium 浏览器扩展，专注于精准量化、分析和提升网页浏览与注意力分配效率。

---

## ✨ 核心功能亮点

### 📊 智能双维度时间追踪

- **前后台精准区分**：毫秒级区分浏览器窗口前台焦点交互（鼠标移动、键盘输入、页面滚动）与后台挂机总时长。
- **无操作空闲检测**：支持自定义开启/关闭无操作检测（超时 30s ~ 10min 自动暂停活跃计时）。
- **子域名归并与黑名单**：智能清洗域名，支持多级子域名合并分析与敏感/开发站点过滤（`localhost`、`127.0.0.1`）。

### 📈 全维度可视化数据看板

- **双色嵌套胶囊对比图**：外层柔和底柱表示总驻留时间，内层电光蓝渐变柱表示实际活跃时间。
- **每日起伏趋势与 24h 分布**：精准分析全天各时段活跃分布热力图与日均专注率。
- **全量明细表与下钻流水**：支持多维度动态排序、全量切页流水日志实时溯源。
- **跨段历史比对**：按选择周期（今日 / 近 7 天 / 近 30 天 / 自定义区间）开展逐日对比。

### 📅 自定义时间区间与吸顶控制

- **下拉弹出式时间选择器 (`DateRangePicker`)**：预设快捷按钮 + 双日期 Picker 弹窗，同一天智能识别 `00:00:00 - 23:59:59`。
- **全局固定吸顶 Header**：顶部标题栏与筛选工具栏在页面滚动时全程 `sticky` 吸顶。
- **可折叠收起侧边栏**：支持收折为极简图标侧边栏，适应更多屏幕尺寸。

### 🔒 100% 本地存储与隐私安全

- **零服务端依赖**：所有切页日志与访问数据完全存储在本地 IndexedDB 数据库中，**绝对不会上传任何第三方服务器**。

---

## 📦 安装与使用指南

1. 下载下方 Assets 中的 **`InsightTrack-v1.0.0.zip`** 压缩包并解压。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 在右上角开启 **「开发者模式」 (Developer mode)**。
4. 点击左上角 **「加载已解压的扩展程序」 (Load unpacked)**，选择解压后的目录即可完成安装！
